ccTools.ProgressDialog

Display a modal progress dialog element in front of the specified component (uifigure, uihtml, uitable and so on).

Syntax

d = ccTools.ProgressDialog(parent)
d = ccTools.ProgressDialog(parent, Name, Value)

Name-Value Optional Arguments

(argument name, type, and possible values or default value)
  • size - scalar text (char | string) - "40px"
  • color - numeric array or scalar text (char | string) - "#d95319"

Example

f = uifigure;
g = uigridlayout(f, 'RowHeight', {44, '1x'}, 'ColumnWidth', {'1x'});
b = uibutton(g);
p = uipanel(g);
